Diese Website ist garantiert frei von XML, XSLT, Anakia, Velocity, Ruby, Perl und weiteren coolen Tools. Ich verwende Eden Kirins genialen Editor ConTEXT um den HTML-Quelltext der Seiten zu bearbeiten (vi ist mir zu frugal und beim Emacs-Examen bin ich durchgefallen).

Mangels fundierter HTML-Ausbildung habe ich bei der Arbeit an Websites praktisch ständig das (ebenfalls geniale) HTML- und CSS-Tutorial SELFHTML im Browser offen. Dessen Hauptautor, Stefan Münz, von dieser Stelle aus herzlichen Dank für diese Werk.

Ich glaube an OpenSource und freie Software - daher verwende ich für gewöhnlich einen Mozilla- oder Firebird Browser (wer tabbed-browsing einmal erlebt hat, wird nichts anderes mehr akzeptieren!).

Das CSS Stylesheet basiert auf einer Vorlage von Verena Segert, die einige sehenswerte Layout-Beiträge für SELFHTML erstellt hat.
