Proxy

Strukturmuster

Proxy stellt einen Platzhalter für eine andere Komponente (Objekt) dar und kontrolliert den Zugang zum echten Objekt.

Beispiel

Das Beispiel zeigt einen Security Proxy. img::proxy_cx.png Nur nach vorheriger Kontrolle durch den DocumentManagerSecurityProxy werden dem Client Zugriffe auf den SimpleDocumentManager gestattet.

Um das Zusammenspiel im Detail beobachten zu können, setzen Sie den log-level auf DEBUG in der logback.xml und führen den zugehörigen TestCase aus.
