Adapter

Strukturmuster

Adapter passt die Schnittstelle einer Klasse an eine von Clients erwartete Schnittstelle an. Das Adaptermuster lässt Klassen zusammenarbeiten, die andernfalls dazu nicht in der Lage wären.

Klassendiagramm

img::adapter_cn.png

Beispiel

Im Beispiel nutzt ein System (TargetSystem), das Personendaten anzeigt, mit einem anderen (SourceSystem) zusammen. img::adapter_cx.png Das zweite System bietet zwar die richtigen Daten, aber in der falschen Form.
Die Anpassung an die von den Clients erwartete Schnittstelle übernimmt der PersonViewAdapter.

Um das Zusammenspiel im Detail beobachten zu können, setzen Sie den log-level auf DEBUG in der logback.xml und führen den zugehörigen TestCase aus. ref::github(),GOF
